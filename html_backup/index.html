What is Quirks Unchained?

Quirks Unchained is a Minecraft Forge Mod, for version 1.19.2, inspired by the popular anime, My Hero Academia!

In this mod, players have access to 1 of 26 unique Quirks, each of which comes with its own unique set of 1 or more abilities.

Quirks Unchained was created using MCreator, a free, easy to use, and highly capable tool for creating mods.
